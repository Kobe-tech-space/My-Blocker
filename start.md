## 本地预览

```bash
npm run serve          # 终端启动
```
或者双击 `start.bat` 一键启动。

浏览器打开 `http://localhost:3000`

## 修改文章后

改完 `.md` 文件后重新构建：

```bash
npm run build
```

## 部署上线

```bash
npm run deploy
```

## 命令速查

| 命令 | 作用 |
|------|------|
| `npm run serve` | 本地预览 |
| `npm run build` | 重新构建 |
| `npm run deploy` | 推送到 GitHub Pages |
| 双击 `start.bat` | 构建 + 打开浏览器 + 启动服务 |
