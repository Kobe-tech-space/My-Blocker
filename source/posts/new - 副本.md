## 为什么开始学习 AI

我是一个大学生，之前更多接触的是传统的编程学习，比如 Java、C++ 和一些基础的软件开发。

最近几年，AI 的发展越来越快：

- ChatGPT
- Claude
- DeepSeek
- Cursor
- Windsurf

这些工具不断出现，让我第一次真正意识到：

> AI 不只是一个聊天机器人，它已经开始改变编程、设计、学习甚至整个互联网。

于是，我决定正式开始学习 AI。

---

## 我的目标

目前我的目标并不是立刻做一个非常复杂的大模型，而是：

1. 学会如何使用 AI 工具
2. 学会 AI 辅助编程
3. 做出属于自己的小项目
4. 学习前后端开发
5. 把项目部署到 GitHub

我希望未来能够：

- 独立完成一个完整项目
- 学会 AI Agent 开发
- 学会网站开发
- 学会 AI 与编程结合

## 我第一次感受到 AI 的强大

以前学习编程时，经常会遇到：

- 环境配置失败
- 看不懂报错
- 不知道下一步该学什么

但现在：

AI 可以帮助我：

- 解释代码
- 生成页面
- 修复 Bug
- 设计项目结构
- 规划学习路线

这让我第一次感觉：

> “一个人也能完成以前需要团队才能完成的事情。”

## 我的第一个 AI 项目

我准备做一个：

# 拼饭 / 社交小项目

项目目标：

- 用户可以发布拼饭信息
- 用户可以查看附近的拼饭
- 支持简单聊天
- 网页端运行
- 后期部署到 GitHub

为了降低难度，我计划：

- 前端：HTML + CSS + JavaScript
- 后端：Java
- AI：使用 DeepSeek-V4-Pro

同时尝试：

- 多 Agent 协作
- AI 自动生成代码
- AI 辅助 UI 设计

## 学习过程中最大的感受

我发现：

学习 AI 最重要的不是“记住多少知识”。

而是：

- 学会提问
- 学会拆解问题
- 学会让 AI 帮你工作

未来真正重要的能力，可能会变成：

> “人与 AI 协作的能力。”
